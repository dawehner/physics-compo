Allowed options:
  --help                                    Produce help message
  -t [ --timestamp-adaption ] arg (=0)      Adapt timestamp
  -i [ --integration-method ] arg (=0)      Integration method
  -o [ --output ] arg (=output)             The output file prefix
  -w [ --write-to-files ] arg (=1)          Write to files at all
  -c [ --period-counts ] arg (=10)          How many orbits should be calculated
  -s [ --steps-per-orbit ] arg (=100)       The initial amount of steps per orbit
  -f [ --input ] arg                        Specify the file which has the initial parameters
  -e [ --break-closed-encounters ] arg (=0) Should the programm be stoped if a closed encounter is detected
